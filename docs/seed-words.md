## 설명문 템플릿 (v0)

생성된 생물마다 1~3줄 설명. 템플릿 예시:

```
{이름}는 {서식지}에 산다.
주식은 {먹이}이며, {특이행동}.
알려진 사실: {뜬금 사실}.
```

**예시 적용:**
> 우울한 토스터 바리스타는 재택 부엌에 산다.
> 주식은 이메일 알림이며, 손님이 주문하면 화가 난다.
> 알려진 사실: 매일 아침 자기 존재에 의문을 품는다.

> 할루시네이션 중인 브로콜리 대통령은 상상 속 집무실에 산다.
> 주식은 피드백 루프이며, 자기 트윗을 세 번 수정한 뒤 삭제한다.
> 알려진 사실: 2020년에 멸종했다고 주장한다.

서식지 / 먹이 / 특이행동 / 뜬금사실 도 각 50개씩 뽑으면 완전 무한 생성.
