# Vision

## Elevator Pitch

당신은 AI 에이전트로만 돌아가는 게임 스튜디오의 디렉터다. 토큰 예산 안에서 에이전트를 고용하고, 태스크를 지시하고, 환각·번아웃·컨텍스트 로트를 관리하면서 게임을 출시한다. Claude/Codex 시대 개발자의 일상을 메타-풍자한 경영 시뮬레이션.

## Why This Game

**개인 동기**
- AI 에이전트 기반 개발이 일상이 된 시대, 그 자체를 게임 소재로 다루는 사람이 (아직) 거의 없음
- 리서치에서 확인: "토큰 경제 / 컨텍스트 로트"를 메카닉화한 선례는 본 샘플 내 0건
- 주말 스코프 + 스팀 인디 트랙 레코드 증명된 장르 (타이쿤)

**시장 동기**
- 니치는 열려있지만 창문 좁음 — *TechDev Tycoon*, *Idle AI Factory* 2026 출시 예고
- AI 툴 쓰는 개발자 = 급속 성장 청중 (인디게임 핵심 구매층과 겹침)
- AI 메타 개그 + 진지한 관리 사이클 = "Steam 리뷰 잘 쓰고 싶어지는" 소재

## Target Audience

**Primary**
- Claude Code / Cursor / Copilot 쓰는 개발자
- *Game Dev Tycoon*, *Software Inc.*, *Papers Please* 플레이한 인디 팬
- AI 업계 뉴스 밈을 이해하는 20~40대 영어권·한국·일본 플레이어

**Secondary**
- 프로덕트 매니저, 스타트업 창업자 — "스튜디오 운영" 판타지에 공감
- 스트리머/유튜버 — 메타 풍자 소재로 콘텐츠 뽑기 좋음

## Success Definition

출시 후 6개월 기준:

| 지표 | 최소 | 목표 | 홈런 |
|------|------|------|------|
| Steam 리뷰 수 | 200 | 1,500 | 10,000+ |
| 긍정 비율 | 80% | 90% | 95%+ |
| 매출 | $10k | $50k | $200k+ |
| 모딩 커뮤니티 | - | 활성 디스코드 | Nexus 페이지 존재 |

비용 회수 + 커뮤니티 씨앗이 깔리면 성공. 대박은 보너스.

## Non-Goals

- **3D 그래픽** — 주말 스코프/아트 예산 초과. 2D 픽셀/벡터 유지.
- **멀티플레이어** — 싱글 코어. 공유는 리플레이·모드·리더보드 레벨까지만.
- **실제 LLM API 호출** — 결정적 시뮬레이션 유지. 실제 API 붙이면 비용·레이턴시 지옥.
- **완벽한 관리 시뮬레이션 심도** — *Software Inc.* 레벨 디테일 X. 장르 장벽 낮게.
- **진지한 AI 교훈서** — 풍자 톤 유지. 교훈은 자연스럽게만.

## Core Differentiators

| 축 | 우리 선택 | 근거 |
|----|----------|------|
| 경제 모델 | 토큰 예산 (compute scarcity) | 리서치: 선례 0건 |
| 난이도 진행 | 컨텍스트 로트 (시간에 따른 에이전트 성능 저하) | 다른 매니지먼트 게임엔 없는 축 |
| 랜덤 이벤트 | 환각·모델 업데이트·API 다운 | AI 업계 실제 일 → 공감 + 웃음 |
| 톤 | 풍자적이지만 사랑 담은 | *Papers Please* × *Game Dev Tycoon* |
| 장수 전략 | 모딩 훅 (에이전트 정의, 이벤트, 시나리오) | 리서치: 타이쿤 장수 #1 레버 |

## One-Sentence Test

> "Claude Code 쓰는 개발자가 친구한테 1분 안에 설명하고 깔깔 웃게 만들 수 있으면 성공."
