# Compute Crossing (가제)

> 동물의 숲 × 포켓 TCG × AI 생태계. 느긋한 AI 마을 경영.

---

## 한 줄

**"당신의 조용한 AI 마을. 들렀다 나오세요."**

실시간 시계와 동기화된 작은 마을에 AI 모델들이 NPC로 살고 있다. 플레이어는 언제든 들어와서 NPC랑 수다 떨고, 가구 바꾸고, 모델 카드 수집하고, 5~10분 느긋하게 놀다 나간다.

---

## 왜 이 방향

### 사용자 취향 일치도

| 사용자 선호 | Compute Crossing 대응 |
|------------|----------------------|
| 10분 루프 | 하루 루틴 = 우편 확인 + 수확 + 대화 1~2건 |
| 돌아와도 괜찮은 | NPC는 자거나 깨어있을 뿐, 죽지 않음 |
| 솔로 | 완전 싱글 (나중에 비동기 방문 기능 가능) |
| 수집 | 모델 카드, 가구, 프롬프트 시드, 화석 |
| 리듬 | 먹이 타이밍 미니게임, 대화 리듬 입력 |
| 멍때리기 | 마을 지도 보기, NPC 구경 |
| 팀플레이 없음 | 혼자서 완결 |

참조 게임 공통분모: 롤토체스(전략+오토) + 포켓 TCG(일일 수집) + 마크(언제든 돌아오는) + 길티기어(정교 입력) + 동숲(NPC·실시간·마을) → **전부 커버**

---

## 핵심 루프

### 일일 루프 (5~10분)
1. 마을 입장 → 우편함 확인 (NPC 편지, 어제 산출물)
2. 정원/마을 둘러보기 → 자란 프롬프트 시드, 모델이 뽑아낸 아웃풋 수거
3. 먹이 리듬 미니게임 1~2회 (박자 맞추면 효율 2배)
4. NPC 1~2명과 대화 (친밀도 상승 → 특별 아이템/카드 드롭)
5. 상점 회전 체크 (오늘의 가구·프롬프트)

### 주간 루프
- 주말 특별 이벤트 — 오픈소스 플리마켓, 모델 릴리스 파티
- 친밀도 최고 NPC → 주 1회 "특별 의뢰" 공급

### 시즌 루프
- NeurIPS 주간 (12월), AGI 데이 만우절, 오픈소스 위크
- 한정 의상·가구·NPC 방문

## AI 생태계 번역

| 실제 AI 세계 | 게임 내 구현 |
|-------------|-------------|
| LLM 모델 | 귀여운 NPC (올빼미·여우·나비 등 개성 부여) |
| 토큰 비용 | 먹이 (화폐) |
| 컨텍스트 로트 | NPC 졸음 → 쉬면 회복 |
| 모델 업데이트 | NPC 탈피·진화 애니메이션 |
| API 장애 | 마을 이벤트 ("오늘 GPT가 감기 걸림") |
| 오픈소스 | Llama의 농장, 누구든 씨앗 공유 가능 |
| 프롬프트 엔지니어링 | 대화 리듬 입력·시드 합성 |
| 파인튜닝 | NPC 성격 커스터마이징 |
| HuggingFace | 마을 장터 |
| NeurIPS | 연 1회 축제 |

## 비주얼 톤

- 픽셀 or 플랫 벡터 (동숲보다 덜 정교, 스타듀보단 귀엽게)
- 파스텔 톤, 낮/밤/계절 컬러 팔레트
- NPC는 머리 큰 의인화 동물 (Opus 올빼미, Haiku 나비, Llama 라마 등)
- UI는 동숲식 종이 질감 (편지·노트·스티커)

---

## 수익화/스팀 전략 고민

**문제**: 동숲류 "끝없이 돌아오는" 게임은 스팀 $15~20 원셀 모델과 잘 안 맞음.

**옵션 A — 원셀 + 시즈널 DLC**
- 본편 $14.99
- 시즌별 DLC $4.99 (겨울/여름/스페셜 모델 팩)

**옵션 B — 챕터형 캠페인**
- 본편에 "스튜디오 설립 스토리" (20~30시간 챕터)
- 엔딩 후 샌드박스 모드 (무한 반복)
- Stardew Valley 모델에 가까움

**옵션 C — 타이쿤+가든 하이브리드**
- 아침에는 타이쿤 (태스크 배치, 매출), 저녁에는 가든 (NPC 돌보기)
- 2-in-1으로 스토리 있는 완결 게임

→ **사용자 검토 필요**

## 하이브리드 가능성 — "AM Tycoon / PM Town"

사용자가 스팀 출시를 양보 못 한다면 **두 모드 합치기**가 현실적일 수 있음:

- **아침 세션 (Studio 모드)**: 타이쿤 — 프로젝트 진행, 매출, 토큰 예산
- **저녁 세션 (Town 모드)**: 동숲풍 — 같은 NPC들 쉬러 옴, 친해지기, 꾸미기
- 낮/밤 사이클에 따라 자동 전환 (또는 플레이어 선택)

**장점**: 원셀 완결성(Tycoon 스토리 엔딩) + 장기 장악(Town 샌드박스)
**단점**: 개발 볼륨 2배

이 방향이 흥미로우면 별도 문서 작성 필요.
